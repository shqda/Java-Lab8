## Основное задание

Реализовать класс `ParallelMatrixProduct` для **многопоточного умножения матриц** `UsualMatrix`.

### Требования

* В конструктор передаётся количество потоков
* Число потоков может быть меньше количества строк первой матрицы
* Матрицы перемножаются параллельно с использованием заданного числа потоков

---

## Сравнение производительности

В `main` необходимо:

* Сгенерировать большие случайные матрицы
* Выполнить:

  * обычное умножение
  * многопоточное умножение
* Сравнить время выполнения

### Примечание

* Для измерения времени использовать методы класса `System` (например, `System.currentTimeMillis()`)

---

## Дополнительное задание

### Параллельный QuickSort

Реализовать многопоточную версию быстрой сортировки:

```java
void quicksort(Object[] array, int threadNum);
```

### Требования

* Поддержка любых типов, реализующих `Comparable`
* Использование нескольких потоков при `threadNum > 1`

---

## Анализ производительности

* Подобрать размер массива, при котором:

  ```
  quicksort(array, threadNum > 1)
  ```

  работает быстрее, чем:

  ```
  quicksort(array, 1)
  ```
